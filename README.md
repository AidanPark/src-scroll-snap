# ScrollSnap

캡처카드를 통해 외부 PC의 화면을 수신하고, 스크롤되는 문서를 자동으로 하나의 긴 이미지로 합성하는 브라우저 기반 도구입니다.

## 개요

다른 컴퓨터(소스 PC)에서 열린 문서를 스크롤하면, 캡처카드를 통해 연결된 내 컴퓨터(타겟 PC)에서 화면을 프레임 단위로 캡처하고, 겹치는 영역을 자동으로 인식하여 하나의 연속된 문서 이미지로 만들어 줍니다.

```
소스 PC (문서 스크롤) → HDMI → 캡처카드(UVC) → USB 3.0 → 타겟 PC (브라우저에서 캡처 & 스티칭)
```

## 주요 기능

- **서버 불필요** — HTML 파일 하나를 브라우저에서 열면 바로 사용 가능
- **실시간 미리보기** — 캡처카드 영상을 브라우저에서 실시간 확인
- **자동 스크롤 감지** — 프레임 간 변화를 감지하여 중복 없이 캡처
- **이미지 스티칭** — 겹침 영역을 매칭하여 하나의 긴 이미지로 합성
- **세로/가로 스크롤 지원** — 문서의 상하 스크롤과 스프레드시트의 좌우 스크롤 모두 대응
- **LLM OCR (예정)** — 멀티모달 LLM API를 활용한 문서 텍스트 추출

## 하드웨어 요구사항

| 장비 | 요구사항 |
|------|----------|
| 캡처카드 | UVC 호환 USB HDMI 캡처카드 (예: Anyport AP-HDC4K) |
| USB 포트 | USB 3.0 이상 권장 |
| 브라우저 | Chrome, Edge 등 Chromium 기반 (getUserMedia 지원) |

## 사용법

### 1. 연결

```
소스 PC의 HDMI 출력 → 캡처카드 HDMI 입력
캡처카드 USB → 타겟 PC의 USB 3.0 포트
```

### 2. 실행

`index.html`을 브라우저에서 엽니다. 별도 설치나 서버 실행이 필요 없습니다.

### 3. 캡처 & 스티칭

1. 캡처카드 장치를 선택하고 미리보기를 확인합니다.
2. 스크롤 방향(세로/가로)을 선택합니다.
3. **[시작]** 버튼을 클릭합니다.
4. 소스 PC에서 문서를 **천천히** 스크롤합니다.
5. **[종료]** 버튼을 클릭하면 자동으로 스티칭이 시작됩니다.
6. 완성된 이미지를 미리보기하고 PNG/JPEG로 다운로드합니다.

> **팁:** 스크롤은 일정한 속도로 천천히 하는 것이 좋습니다. 너무 빠르면 프레임 간 겹침이 부족하여 매칭에 실패할 수 있습니다.

## 기술 스택

- **영상 입력:** `getUserMedia()` API (브라우저 네이티브)
- **프레임 처리:** Canvas 2D API
- **이미지 매칭:** NCC (Normalized Cross-Correlation) 자체 구현
- **UI:** React + Tailwind CSS (CDN)
- **외부 의존성:** 없음
